Reserves a dataset with annotated domain information for parallel access.

Parameters:

sizeintended 3D dimension for local dataset
globalSizeif not NULL, returns the global size of the dataset
globalOffsetif not NULL, returns the global offset for the calling process
rankmaximum dimension (must be between 1-3)
typetype information for data. available types must be implemented by concrete DataCollector
namename for the dataset, e.g. 'ions'
